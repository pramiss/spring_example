Spring framework vs Spring boot

<Spring 책 추천>
Spring 3.0, 3.2 => 토비의 스프링 (입사 후 코드에 익숙해진 후 읽어보기)

<Request  ->  Response>
                   @Controller + @ResponseBody : return String -> HTML
                   @Controller + @ResponseBody : return Map -> JSON
                   @Controller + @ResponseBody : return (사용자 객체) -> JSON
                   @RestController = @Controller + @ResponseBody

<HttpMessageConverter : 중간 과정>
@RestController + return String -> HttpMessageConverter(클래스) -> HTML
@RestController + return 객체(Map, List, 사용자 객체) 
                        -> HttpMessageConverter(클래스) -> JSON

HttpMessageConverter의 역할은 나중에 Spring Framework를 할때 도움됨
