Q. MySQL에 Date 타입에 데이터를 넣으려면 Spring에서 어떤 자료형을 넣어야할까?
A. LocalDate, Date, String 모두 가능하다.
애초에 Workbench에서 데이터 삽입을 할때 '2024-00-00'과 같이 String으로 넣었음


<6강에서 요청방식이 하나 더 추가(AJAX)>
GET : query string
- 브라우저
- a태그
- from 태그 method="get" -> 전송 후 반드시 화면 이동
- 리다이렉트
- js location.href
+ AJAX 통신

POST : request body
- form 태그 method="post" -> 전송 후 반드시 화면 이동
+ AJAX 통신

<form의 흐름>
form -> 브라우저(파라미터 취합) -> 서버 -> 화면이동

<Ajax의 흐름>
form -> 내가 파라미터 취합 -> 서버 -> 화면이동
: 브라우저가 하던 일을 "내"가 하게 됨
: 실제로 form보다 Ajax를 더 많이 사용 -> 이유: input값을 검사하는 경우가 많으므로
: 웹페이지의 모든 데이터를 다 가져오지 않고, 필요할때 가져옴. -> 동적으로 웹페이지 구현가능
: 비동기 방식


    			// AJAX 통신 => 서버에 요청
    			// 딕셔너리로 넘긴다. (jQuery 문법임)
    			$.ajax({ // 시작과 끝이 ajax임
    				// request
    				type:"POST"
    				, url:"/lesson06/ex01/add-user"
    				, data:{"name":name, "yyyymmdd":yyyymmdd, "email":email, "introduce":introduce}
    				// Ajax가 알아서 Get(QueryString), Post(RequestBody)으로 보냄
    				
   					// Post 요청 후 다시 여기로 돌아옴(callback 함수)
   					// ex) Post가 새로운 데이터 insert라면
   					// AJAX -> (url)Controller -> BO -> Mapper -> DB -> Mapper -> BO -> Controller -> AJAX
    			
    				// response
    				// callback 함수
    				
    				// request 성공시 여기로 도달
    				, success:function(data) { // data파라미터: 응답값(String 항상!!, Controller에서 return되는 값을 의미)
    					// alert(data);
   						if (data == "성공") {
   							location.href = "/lesson06/ex01/after-add-user-view";
   						}
    				}
    				, error:function(xhr, status, error) {
    					alert(xhr);
    					alert(status);
    					alert(error);
    				}
   					
    			}); //-- ajax
